## 커멘드
`/cry1` : 누니머기의 울음소리를 출력합니다.
`/cry2` : 누니머기의 울음소리를 출력합니다.
`/cry3` : 누니머기의 울음소리를 출력합니다.
`/truefact` : 누니머기의 진실을 출력합니다.
`/falsefact` : 누알못들의 거짓부렁을 출력합니다.
